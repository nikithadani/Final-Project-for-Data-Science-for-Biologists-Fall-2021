Introduction

I obtained the Breast Cancer cell data from the Kaggle website here, which also directs you to the UCI Machine Learning Repository here. The dataset contains 569 samples (357 benign and 212 malignant) of different features of breast tumor cell nuclei by using the biopsy method of fine needle aspirate (FNA) of a breasts mass. I picked this data set mainly because I actually wanted to learn what measurements are taken into consideration when doing biopsy on breast mass, I had no idea what exactly helps determine whether it is a malignant or benign breast tumor. Quite honestly by doing the model to predict future diagnosis, makes me feel successful in a way, even though my model is not being used for such important diagnosis, I feel like I can actually do models to keep pushing for better healthcare systems.

The attributes in the data set are as follows:

  • id: Identifies the sample
  • diagnosis: Tumor is 1= Malignant or 0= Benign
  • radius: Mean of distances from center to points on the perimeter
  • texture: Standard deviation of gray-scale values
  • perimeter: Distance around nuclear border
  • area : Area of nucleus
  • smoothness: Variation in the cells radial lengths
  • compactness: perimeter\(^2\) / area)
  • concavity: Size of indentations in the nuclear border
  • concave points: Number of concave portions of the nuclear border
  • symmetry: Deviation of nuclei shape from ideal
  • fractal dimension: Measurement of nuclear border irregularity or coastline approximation
  • The mean(variable_mean), standard error(variable_se) and worst or largest (mean of the three largest values, shown as variable_se) of these features were computed for each image, resulting in 30 features. All feature values are recorded with four significant digits

Question 1: Modeling

Do these specific variables explain variation in/predict status of response?

Methods

model_selection <- glm(diagnosis ~ radius_mean 
                       + texture_mean 
                       + perimeter_mean 
                       + area_mean 
                       + smoothness_mean 
                       + compactness_mean 
                       + concavity_mean 
                       + symmetry_mean 
                       + fractal_dimension_mean,
                       data = new_can,
                       family = "binomial")

model_formula <- as.formula(model_selection)
model_formula
## diagnosis ~ radius_mean + texture_mean + perimeter_mean + area_mean + 
##     smoothness_mean + compactness_mean + concavity_mean + symmetry_mean + 
##     fractal_dimension_mean
model_roc <- roc(new_can$diagnosis,
                 model_selection$fitted.values)  #MODEL!
model_roc$auc  #extract auc    #0.9866
## Area under the curve: 0.9866
#Logic curve 

tibble(logistic_x = model_selection$linear.predictors, 
       logistic_y = model_selection$fitted.values, 
       truth = if_else(new_can$diagnosis == 1, "Malignant", "Benign")) -> logistic_model_plot_data


ggplot(logistic_model_plot_data) + 
  aes(x = logistic_x,
      y = logistic_y) + 
  geom_line() + 
  geom_point(aes(color = truth), alpha = 0.4) +
  scale_color_viridis_d(option = 'plasma') + 
  labs(x = "Linear predictors", 
       y = "Predicted probability of breast cancer", 
       color = "True disease status",
       title = "Logistic Curve") -> modelfig
modelfig
